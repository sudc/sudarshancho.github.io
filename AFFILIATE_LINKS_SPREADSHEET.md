## Instructions for Use:

1. **Sign up for affiliate networks** listed in the first table
2. **Update Account ID and API Key** once approved
3. **Generate affiliate links** for each destination/service/product
4. **Replace PLACEHOLDER** with actual affiliate URLs
5. **Update Status** from "Pending" to "Active" once link is live
6. **Track performance** by updating clicks, conversions, and revenue columns
7. **Review monthly** to identify top-performing links and categories
8. **Optimize** by focusing on high-converting platforms and destinations

## Link Generation Tips:

### Hotels:
- Use destination-specific search URLs
- Include check-in/check-out date parameters (flexible dates)
- Add number of guests and rooms if applicable
- Example: `https://www.booking.com/searchresults.html?city=-2092174&aid=YOUR_AFF_ID`

### Flights:
- Create route-specific search links
- Use flexible date parameters
- Example: `https://www.cleartrip.com/flights/results?from=BOM&to=DEL&aid=YOUR_AFF_ID`

### Health:
- Link to specific test packages
- Use category pages for medicine orders
- Example: `https://www.healthians.com/packages/full-body-checkup?aff=YOUR_AFF_ID`

### Insurance:
- Create type-specific comparison pages
- Include trip duration for travel insurance
- Example: `https://www.policybazaar.com/travel-insurance/?aff=YOUR_AFF_ID`

### Deals:
- Use category-specific pages
- Link to bestseller or deal pages
- Example: `https://www.amazon.in/electronics?tag=YOUR_AFF_TAG`

## Notes:
- Cookie Duration: The time window for which a click is tracked
- Commission: Percentage or fixed amount earned per conversion
- Status: Pending → Active → Paused → Inactive
- Update this sheet regularly to track performance
- Use UTM parameters for better tracking in Google Analytics
